# Early-Childhood-Teacher-Library

I build up this library to collect handful and practical activities and resources for early childhood teachers and educators. 

This include general topics for any age groups and those classified in terms of age and learning stages. 

Since babies (0-1) mostly develop on sensory, caring and simple reactions to the world and carers, there is no need to provide flaring activities. It is suggested to grab some ideas from [EYLF](https://www.ectplanning.com/activities) and extend activities build on children's interets. 

Our library only collect those for [toddlers](./Toddler.md) (1-2year old) and [kinders](./Kinder.md) (3-5). 

## Description of the files in the repository

I intend to make this repository into a useful webpage that has a better visual effect. **index.md** is the home page of my GitHub page website. **_config.yml** is the configuration file that automatically sets up the webpage based on the existing Jekyll theme styles. The original documents of the repository are **Kinder.md** and **Toddler.md**, which contain the main text of my library (repository).
