# Early-Childhood-Teacher-Library

I build up this library to collect handful and practical activities and resources for early childhood teachers and educators. 

This include general topics for any age groups and those classified in terms of age and learning stages. 

Since babies (0-1) mostly develop on sensory, caring and simple reactions to the world and carers, there is no need to provide flaring activities. 

Our library only collect those for toddlers (1-2year old) and kinders (3-5).
